## Error codes

General
1: incorrect number of arguments

Preprocessor
2: unable to read file
3: something went wrong while reading file
4: max macros limit reached

Memory Allocator
5: unable to allocate memory

Parser
6: unexpected token (most likely a syntax error, top-level)
7: declaration error (most likely a syntax error)
8: mismatched token (token check failed)
9: function argument with invalid or no type
